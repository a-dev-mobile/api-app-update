
Запуск контейнеров:
make up

Остановка контейнеров:
make down

Пересборка контейнеров:
make rebuild

Просмотр логов:
make logs

Открытие оболочки в контейнере
make shell

Очистка неиспользуемых данных:
make clean

Просмотр состояния контейнеров:
make ps

Проверка и удаление неиспользуемых данных:
make prune

Перезапуск контейнеров:
make restart
